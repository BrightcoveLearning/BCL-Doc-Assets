Developer Tools and Concepts

Brightcove Learning Services offers a set of tools and concepts to help developers work with the Brightcove Player, APIs, and SDKs.

Developer topics include conceptual guides to working with JSON data, CORS, and HLS video, and debugging apps and pages on the desktop or mobile devices using the developer tools in the browser. There is also a full guide to setting up a client-proxy app that can access the Brightcove APIs from a web interface.

There is an overview of testing tools for API requests, as well as an online testing tool, a guide to using the popular REST client Postman, and setting up cURL to test requests. There are also a couple of utilities for encoding strings and converting date formats.
