When creating a Dynamic Ingest request using the Dynamic Ingest API or uploading videos using the Upload module, you must supply a profile which determines the number and type of renditions created by the Video Cloud ingestion process. This topic provides detailed information on each of the Dynamic Delivey profiles. For non Dynamic Delivery accounts see these Standard Ingest Profiles instead.

It is also possible to create your own custom ingest profiles. For information, see Managing Ingest Profiles.
