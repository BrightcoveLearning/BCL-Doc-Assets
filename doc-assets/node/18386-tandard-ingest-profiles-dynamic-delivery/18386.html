When creating a Dynamic Ingest request using the DI API or uploading videos using the Upload module, you must supply a profile which determines the number and type of renditions created by the Video Cloud ingestion process. This topic provides detailed information on each of those profiles.

Note: When lower quality source content is uploaded and one of the standard profiles is selected, Video Cloud will only produce renditions that are equal to or lower in quality than the source.

It is also possible to create your own custom ingest profiles. For information, see Managing Ingest Profiles.
