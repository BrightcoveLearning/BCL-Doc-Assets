Data Collection API

The Data Collection API allows you to send analytics data for your videos to the Brightcove Analytics system.

Note that data collection for Brightcove web and native device players is built into the players and automatic. You only need to use the Data Collection API if you are NOT using a Brightcove player for your videos.
