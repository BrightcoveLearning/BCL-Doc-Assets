Introduction

Zencoder supports high-dynamic-range (HDR) video, initially adding options to produce HDR10 outputs from HDR color-graded masters.

In addition to this topic, you should also look at HDR10 Video Output for additional information and specific video encoding settings that are required for HDR10 outputs.

API Versions: V2

Default: none

Valid Values: 0.0001-0.7400
