In-Page Experiences Platform and Client APIs

The In-Page Experiences platform API gives you programmatic control over In-Page Experiences, allowing you to manage them within your own CMS system or app.

The In-Page Experiences client API allows you to interact with an In-Page Experience at runtime, in the same way the Brightcove Player API is used to control the Brightcove Player.
