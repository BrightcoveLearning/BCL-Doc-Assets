Introduction

The Dynamic Ingest API is not just for adding new videos to your Video Cloud account. After you ingest a video, there are a number of ways you can use the API to modify or add assets to the video. These operations are summarized in the sections below.

Replace the video

You can replace the video with a new version, using the same or a different ingest profile

Retranscode the video

You can retranscode the video to create a new set of renditions, keeping the existin thumbnail and poster images, or capturing new ones.

Add images

You can add the video with a new version, using the same or a different ingest profile
