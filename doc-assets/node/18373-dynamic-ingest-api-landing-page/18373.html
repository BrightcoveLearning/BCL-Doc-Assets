Dynamic Ingest API

Brightcove's Dynamic Ingest (DI) API is used to ingest videos and other media assets into Video Cloud. Which version of the docs you should use depends on whether your account is set up for Dynamic Delivery or the legacy ingest system. If you are not sure about that, here's how you can tell:

  • If the account was created on or after October 17, 2017, the account is on Dynamic Delivery
  • If you go to the Upload module in Studio, expand the Ingest Profile dropdown menu, and see Dynamic Delivery profiles, your account is on Dynamic Delivery:
  • If neither of the above is true, your account is on the legacy ingest system
