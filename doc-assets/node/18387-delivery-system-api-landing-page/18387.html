Delivery System API

The Delivery System API allows you to manage and deploy of a group of files, called a repository. This system was setup as a storage area for player related files, like the JavaScript and CSS files for custom built plugins.

Note that most users don't need to use this API. It is intended for experienced developers who have a good understanding of Git.
