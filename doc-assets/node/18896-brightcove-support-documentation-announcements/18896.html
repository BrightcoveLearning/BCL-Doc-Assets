New option for creating live jobs in the Live API

13 May 2019

The color_full_width has been added to outputs settings for creating a Live job. In most cases you won't need to use it, but specifying color_full_width as true will force a color range conversion of input video file to "full", if it is "limited". By default, the encoder will not try to change the range.
