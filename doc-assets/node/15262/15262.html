The Brightcove SDKs support the following methods for live playback:

  • Live streaming
  • DVR Live streaming

Live streaming

The classic approach to bringing live content to viewers, Live Streaming provides a continuous stream that viewers watch in real-time.

Feature iOS SDK Android SDK
Live stream
Supports    
Timed metadata/ ID3 tags

 

Note: The Android SDK supports ID3 metadata tags when using the BrightcoveExoPlayer view.

Looking for a way to deliver high quality Live streams? Check out Brightcove’s Live Module and Delivering Live Streams documentation.

DVR Live streaming

DVR Live goes above and beyond the classic approach to Live Streaming by allowing viewers to take control of their viewing experience. Providing the viewer full control over playback, DVR Live enables your audience to seek to past moments in a VoD styled setting. Then, with the click of a button, seamlessly jump back to the live action to continue watching in real time.

Feature iOS SDK Android SDK
DVR Live stream

 

Looking for a way to deliver high quality DVR Live streams? Check out Brightcove’s Delivering Live Streaming Video with DVR documentation.

Note: The Brightcove Player SDK for Android supports Live Streaming and DVR Live Streaming. However, there are some limitations with playing back HLS files. For details, see the HLS Limitations without Brightcove's Native SDK document.

Looking for developer docs? Check out the Brightcove Native Player SDK for iOS and the Brightcove Native Player SDK for Android documents.
