Overview

When you build Apple TV apps using the Brightcove Native SDK for tvOS, you will get a set of playback controls to allow users to scrub through video content, change the view size, get video information, and manage captions and audio settings.

The controls provided by the Brightcove Native SDK for tvOS include the following:

  • Progress bar with current time and end time
  • Top menu for video info, captions and audio
  • Ability to add your own custom menu buttons and functionality

The controls and gestures used will be familiar to users, since they are similar to those found in the AVKit interface.

With Brightcove Server-Side Ad Insertion (SSAI), users will see a custom look during ad playback.

For details about using the Native SDK, see the Overview for the Brightcove Player SDK for tvOS document.

Supported versions

Apple TV UI controls are available with the following versions:

Device OS version

tvOS 11+

Native SDK version

Brightcove Native SDK for Android version 6.3.0+

Playback controls

The playback controls are manipulated using gestures and button presses on the Apple TV Remote. Below are some of the commands and actions you can use with your app. For details, see the following:

  • Built-In TV Player Controls for tvOS
  • Using the TV Player UI With The Brightcove Player SDK for tvOS

Progress bar

A single tap on the Apple Siri Remote track pad will show/hide the progess progress bar. When playback is paused, panning left and right will scrub backward and forward through your video.

The user can click and hold, and then pan left or right to fast forward or reverse through a video.

While the video is playing, the user can double click on the left quarter of the Siri remote track pad to jump back 10 seconds or double click on the right quarter of the track pad to jump forward 10 seconds.

Video view size

A double tap on the Apple Siri Remote track pad will cycle through 3 standard video view sizes:

  • Full screen with aspect ratio respected (video may be cropped)
  • Letterboxing to respect aspect ratio (full video displayed)
  • View stretched to fit full screen (aspect ratio ignored)

Tab bar

Swiping down on the Apple TV Remote track pad will display the top menu and hide the progress view. Once visible, swiping up will hide the top menu.

The top menu has the following default items:

  • Info
  • Subtitles
  • Audio

You can modify the text labels associated with each top menu item, or you can create your own custom top menu items.

Info

Selecting the Info top menu item displays the title, description and length associated with the currently loaded video. This data is retrieved from the video's metadata in the Playback API.

Subtitles

Selecting the Subtitles top menu item displays options for showing captions associated with the video. Here, you can manage the subtitles/captions to show during playback.

Audio

Selecting the Audio top menu item displays the audio options available during video playback.
