Limitations

Although native players such as Windows Media Player, Flash Player and Quicktime support deep linking through different mechanisms, this behavior is not directly exposed through the HTML video or source elements. It is, however, indirectly exposed through the currentTime property of the video element. This effectively allows HTML players like the Brightcove player and YouTube to layer deep linking support on top of native video players.

Social media use

As mentioned above, HTML players can support deep linking, but this necessitates that the player be either embedded in the page or included as an iframe. Unfortunately, most social services do not follow this practice, and instead, substitute a native player which does not support deep linking. As of July 2014 only one major social network, Twitter, supports direct inclusion of the Brightcove player in the social feed using an iframe, making deep linking possible.

For details about deep linking with Twitter, see the Social Media Plugin document.

Specify deep linking

To use a deep link in a Brightcove player, specify an offset using a format popularized by YouTube. It follows the format: 00h00m00s, where each of the three parts represents a different unit of time (hours, minutes, seconds). Each of the parts must be present, so to offset 15 seconds you would write: 0h0m15s. This is then added to the URL as a query string: ?t=0h0m15s. An example of a deep link to a video follows:

https://players.brightcove.net/1752604059001/Sy4QijlIf_default/index.html?videoId=4172255216001#t=00h00m15s

Using with videoId

When you specify a videoId as a query parameter in the Brightcove player URL, you will use the # character (a fragment identifier) in front of the time parameter. Here is an example of the query string:

?videoId=4172255216001#t=00h00m15s

Using with autoplay

You can even include the autoplay parameter to set the player to start playback automatically. Notice that the & symbol is used between the time and autoplay parameters:

?videoId=4172255216001#t=00h00m15s&autoplay=true

Using with iframe implementation

To start playback of a video at a specified time with an iframe implementation, add query parameters to the Brightcove player's src property as follows:

<iframe src="//players.brightcove.net/1752604059001/Sy4QijlIf_default/index.html?videoId=4172255216001#t=00h00m15s&autoplay=true"
      allowfullscreen
      webkitallowfullscreen
      mozallowfullscreen
      width="640"
      height="360"></iframe>
