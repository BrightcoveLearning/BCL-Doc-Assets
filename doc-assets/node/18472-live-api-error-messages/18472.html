Error message reference

Live API Error Messages
Error Response Body Description Applies to
400 BAD REQUEST 
{
	"error": "Invalid region"
}
 The region specified in the request does not exist or is not supported Create live job
400 BAD REQUEST 
{
	"error": "Invalid input value"
}
 The JSON included an invalid value for some field Create live job, create VOD clip, create/update ad configuration, create/update beacon set, ingest slate, create beacon set
400 BAD REQUEST 
{
	"error": "The notification target type is not supported currently"
}
 The notifiction url was missing the protocol or the protocol is not a supported type
401 UNAUTHORIZED 
{
	"message": "Unauthorized"
}
 The X-API-KEY header was not sent with the request, or the key value was not valid All requests
500 INTERNAL SERVER ERROR 
{
	"error": "After filtering, there is no job to process"
}
 You attempted to stop a live job that has already been stopped Cancel live job
500 INTERNAL SERVER ERROR 
{
	"error": "DB getItem, no results found"
}
 You specified a job id that does not exist Get job details, cancel live job, activate SEP job, insert cuepoint, create VOD clip
