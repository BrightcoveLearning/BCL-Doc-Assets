Introduction

The Adobe-AEM-Brightcove Connector allows you to manage Brightcove Video Cloud videos and players within AEM, and easily embed videos in AEM pages.

Note that this guide uses AEM 6.3. The connector also works with AEM 6.2, and the steps are similar.

Installation steps

To install the connector, follow these steps:

  1. Verify that you have FFmpeg installed by opening a command line and typing: 
    ffmpeg
  2. If the ffmpeg command is not found, you need to install it:
    • Install on Mac
    • Install on Windows
    • Install on Linux
  3. Set up a dev instance, if you do not already have one, with: 
    java  -Xmx1400M -jar AEM_6.3_Quickstart.jar
