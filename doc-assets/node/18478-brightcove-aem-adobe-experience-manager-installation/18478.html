Introduction

The Adobe-AEM-Brightcove Connector allows you to manage Brightcove Video Cloud videos and players within AEM, and easily embed videos in AEM pages.

Note that this guide uses AEM 6.3. The connector also works with AEM 6.2, and the steps are similar.

Installation steps

To install the connector, follow these steps:

  1. Verify that you have FFmpeg installed by opening a command line and typing: 
    ffmpeg
  2. If the ffmpeg command is not found, you need to install it:
    • Install on Mac
    • Install on Windows
    • Install on Linux
  3. Set up a dev instance, if you do not already have one, with: 
    java  -Xmx1400M -jar AEM_6.3_Quickstart.jar
  4. If you use this Quickstart, your AEM instance will be hosted at: http://localhost:4502 by default. The following sections will assume this default location, so if you are using a different location or port, you will need to adjust the URLs shown in the steps, or use the AEM navigation menu instead.

Installing the connector

The following steps will walk you through installing the connector.

  • Go to https://github.com/coresecure/Adobe-AEM-Brightcove-Connector/releases and download the latest version of the connector (the ...SNAPSHOT.zip file).
  • Start your dev instance of AEM.
  • After you login, go to the CRX Package Manager (http://localhost:4503/crx/packmgr/index.jsp)
