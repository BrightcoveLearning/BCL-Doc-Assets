Special highlighted text blocks

We have 3 kinds of special highlighted text blocks - they are most effective if you use them sparingly! You can put any HTML you like inside them.

Information block - for just information you want to call out.

This feature is currently in beta - if you are interested in participating in the beta, contact....

Tip block for best practices kinds of things.

If you are constructing a request body for API requests, do not manually create the JSON string. Instead create a native data structure and use your language's JSON.stringify() method (method names vary) to convert the data into JSON.

Warning block for gotchas and things you should not do!

If you are creating a data structure for a Dynamic Ingest request, note that the capture-images field name is hyphenated, and languages like JavaScript do not allow you to use dot notation with hyphenated property names:

  • requestBody.capture-images = true; this is wrong!
  • requestBody['capture-images'] = true; this is correct
