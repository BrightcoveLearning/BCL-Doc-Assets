Introduction

The CMS API has a feature to delete the digital master (source files) in their Video Cloud catalog. This will permit customers to perform one-time cleanup tasks to decrease the size of their managed content.

Requirements

The delete digital master operation will work only for videos ingested using the Dynamic Ingest API or the Upload Module. Deletion of digital masters is not supported for videos ingested using legacy ingest methods.
