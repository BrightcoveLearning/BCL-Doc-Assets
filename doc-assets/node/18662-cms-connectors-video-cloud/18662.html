CMS connectors provide integration between several popular CMS systems and Video Cloud. They allow CMS users to manage Video Cloud videos and players, and embed video players in documents without going to Video Cloud Studio or writing code. And since the connectors handle most of the integration work, there is little if any code for you to write to get them installed and configured for your CMS.
