  • Inlet Spinnaker 8100
  • Cisco Media Encoder AS7100
  • Cisco Media Encoder AS8100
  • Digital Rapids TouchStream
  • Digital Rapids StreamZHD ABR Live
  • Digital Rapids DRC-6550 Card
  • Viewcast Niagra 4100
  • Viewcast Niagra 7550
  • Viewcast Niagra 9100
  • Haivision Kulabyte
  • Envivio 4Caster
  • Elemental Elemental Live
  • DVEO Mini
  • DVEO Midi
  • DVEO Maxi

When delivering live events using the Brightcove Player, we recommend using the settings defined by the videocloud-default-v1 ingest profile.
