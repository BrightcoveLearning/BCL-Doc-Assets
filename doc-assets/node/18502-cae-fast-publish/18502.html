Introduction

Fast publish allows you to use CAE and still get videos online as quickly as possible by creating one or more static renditions which can be played immediately. Once all the dynamic renditions have been processed, the static rendition(s) will be deleted automatically.

Note that Context Aware Encoding is in beta. If you are interested in participating in the beta, please contact your Account Manager.

Requirements

In order to use fast publish, you must:

  1. Have an account enabled for Dynamic Delivery
  2. Have the account enabled for CAE
  3. Create a custom CAE ingest profile using the (instructions below)
