When creating live events using the Live module or Live API, you must select a Live ingest profile which determines the number and quality of the renditions created by Brightcove Live. This topic provides detailed information on each of the Live ingest profiles.

Note: These profiles are used specifically when creating live events. They are not used when uploading videos to Video Cloud.

It is also possible to create your own custom Live ingest profiles. For information, see aaa.
