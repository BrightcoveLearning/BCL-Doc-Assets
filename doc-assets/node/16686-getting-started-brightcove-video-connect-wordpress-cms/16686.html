The Brightcove Connector for WordPress is an open source plugin.  You are free to install, use and modify it for your needs.  If you are unable to successfully install the connector on your WordPress server, Brightcove offers a CMS Connector onboarding package which provides direct access to the connector developers for installation support.  Your Brightcove Account Manager can provide more details on the onboarding package.

If the Brightcove Connector is installed but not working as documented, please fill out this support worksheet and include it with a support ticket to Brightcove customer support.
