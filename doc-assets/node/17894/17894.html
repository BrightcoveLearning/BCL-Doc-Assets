Overview

You specify the API version you want to use through the URL your application uses to post to Zencoder - posts to https://app.zencoder.com/api/v2/ use V2. If no version is specified the request will use the edge API where things can change without warning, so only use this if you're experimenting with a new feature before it's officially available, never in production.

API versions will be closed down as new versions come online, and we will always give at least 6 months notice before a version is shuttered so that you have time to migrate your application.
