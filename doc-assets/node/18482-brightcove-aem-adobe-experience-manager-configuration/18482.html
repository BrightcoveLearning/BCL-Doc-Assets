Introduction

The Adobe-AEM-Brightcove Connector allows you to manage Brightcove Video Cloud videos and players within AEM, and easily embed videos in AEM pages.

Note that this guide uses AEM 6.3. The connector also works with AEM 6.2, and the steps are similar.

Configuration steps

The following sections will walk you through the steps to configure the Adobe-AEM-Brightcove Connector. Note that before completing these steps, the connector must be installed - see Adobe-AEM-Brightcove Connector: Installation

Disable DAM transcoding

Unless you need DAM transcoding for non-Brightcove videos, you should turn it off to save processing time and storage:

  2. Select Workflow:
  3. Select Models:
  4. Select List View:
  5. Select DAM Update Asset:
  6. Select Edit:
  7. Double-Click FFmpeg Transcoding from the workflow to edit it:
  8. Select the Process tab in the Step Properties dialog:
  9. Change the Process to No Operation:
  10. Change Ok to close the dialog and then save the changes
