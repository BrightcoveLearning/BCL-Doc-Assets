Introducing playlists

A playlist is a collection of videos that are grouped together in a particular order for playback in a Video Cloud player. The Media module makes it easy to select the videos that make up a playlist and then publish the playlist. You can create two types of playlists:

  • Manual - Videos are manually added to the playlist
  • Smart - Videos are added to the playlist automatically, based upon video tags

Note: A maximum of 100 videos can be added to a playlist.

The following video tutorial will give you an overview of creating playlists.

 

Creating a manual playlist

You use the Media module to create playlists. To create a manual playlist:

  1. In the side navigation panel, click the Create Playlist link.
  2. Set the playlist type to Manual.
  3. Enter a Playlist Name.
  4. Check the Show Playlist in Sidebar option to have the playlist appear in the sidebar.
  5. Click Save.

Once the playlist is created, select videos to add to the playlist and then click Add to > Add To Playlist.

Creating a smart playlist

To create a smart playlist:

  1. In the side navigation panel, click the Create Playlist link.
  2. Set the playlist type to Smart.
  3. Enter a Playlist Name.
  4. Check the Show Playlist in Sidebar option to have the playlist appear in the sidebar.
  5. Enter the tags that will be used to define the videos that will appear in the playlist.
  6. Select a play order to determine the order that the videos will be played.
  7. Click Save.

Hands-on Task: Creating playlists

In this task you will create manual and smart playlists.

Create a manual playlist

  1. In Video Cloud Studio, click Media to open the Media module.
  2. In the side navigation panel, click the Create Playlist link. The New Playlist dialog will open.
  3. Set the Playlist type to Manual.
  4. Set the Playlist Name to Favorites.
  5. Check Show Playlist in Sidebar to have the playlist appear in the sidebar.
  6. Click Save.

Add videos to the manual playlist

  1. Click All Videos in the side navigation panel to display all the videos in the account.
  2. Click the checkbox next to videos you want to add to the playlist.
  3. Click Add To > Add To Playlist.
  4. Select the Favorites playlist and then click Add.
  5. Click the Favorites playlist in the side navigation panel and confirm the videos have been added to the playlist.
  6. Rearrange the order of the videos by clicking on a row and dragging it to a new position in the list.

Create a smart playlist

  1. In the side navigation panel, click the Create Playlist link.
  2. Set the Playlist type to Smart.
  3. Set the Playlist Name to Birds.
  4. Check Show Playlist in Sidebar to have the playlist appear in the sidebar.
  5. Set the Tags field to contains one or more and enter birds for the tag.
  6. Click Save.
  7. Click the Birds playlist in the side navigation panel and confirm the bird videos have been added to the playlist.
