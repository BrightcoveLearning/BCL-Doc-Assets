Player plugins use a combination of HTML, JavaScript and/or CSS to customize the behavior of the player. Plugins can be developed to:

  • Modify the default behavior of a player
  • Add functionality to a player
  • Customize player appearance
Adding plugin functionality to your Brightcove player is available for Video Cloud Pro and Enterprise customers. If you are interested in upgrading your Video Cloud account, please contact Brightcove for more information.

To configure plugins for the player, follow these steps:

  1. Click Players in the navigation header to open the Players module.
  2. Click the link for the player you want to configure.
  3. Locate the Plugins section and click Edit or double-click in the section.
  4. Click JavaScript and provide a URL to the file that contains the plugin code. Click the + button to add additional files.
  5. Optionally, click CSS and provide a URL to the file(s) that contains the CSS code used by the plugin. Click the + button to add additional files.

    Note: Be sure you supply valid CSS. If you supply non-valid CSS, you will see an error in the preview window stating Unable to find file [dist/bc.min.css]. Please check your player configuration.

  6. Click Name, Options (JSON) and provide the name of the plugin (defined in the JavaScript file). The options text area can be used to paste in JSON formatted options. The options usage is shown in the plugin JavaScript code. Click the + button to add additional plugins.
  7. Click Save to save the changes.

Note: Any changes made to a player may take up to 5 minutes to appear in published players.

For information on creating plugins, see the Customize a Player section of the Quick Start to Player Management developer document.
